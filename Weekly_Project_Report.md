# AI-Powered Public Infrastructure Monitoring System: 16-Week Project Report

This table breakdown maps the development of the AI-Powered Public Infrastructure Monitoring System across two semesters, spanning from November 2025 to March 2026.

| Date | Activity Plan | Execution | Remark by Guide |
| :--- | :--- | :--- | :--- |
| **WEEK 01**<br>(3rd Nov to 8th Nov) | Project Selection and Domain Research | Finalized the project domain: Smart City and Infrastructure. Researched existing manual systems and identified key areas for automation (Streetlights, Water, Roads). | |
